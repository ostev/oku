-   They denote the start and end of a <DocLink symbol="function" tooltip={false}>function's</DocLink> <DocLink symbol="parameter" tooltip={false}>parameters</DocLink>: `forward(1)`
-   They denote the start and end of an <DocLink symbol="if" tooltip={false}>`if`</DocLink> <DocLink symbol="statement" tooltip={false}>statement's</DocLink> condition: `if (x === y) {}`
-   They can be used to order arithmetic operations: `(5 + 3) * 3 !== 5 + 3 * 3`
